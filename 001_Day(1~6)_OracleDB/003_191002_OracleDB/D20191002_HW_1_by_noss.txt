## A. 조인 실습 문제 SQL 92 -> 99 표준안으로 

1. 사원들의 이름, 부서번호, 부서이름을 출력
SELECT E.ENAME, DEPTNO, D.DNAME
FROM EMP E NATURAL JOIN DEPT D

2. 30번 부서의 사원들의 이름, 직업 부서위치 출력
SELECT E.ENAME, E.JOB, D.LOC
FROM  EMP E NATURAL JOIN DEPT D
WHERE DEPTNO = 30

3. 커미션을 받는 사원의 이름, 부서이름 및 부서위치를 출력
SELECT E.ENAME, D.DNAME, D.LOC
FROM EMP E NATURAL JOIN DEPT D
WHERE E.COMM IS NOT NULL

4. DALLAS에서 근무하는 사원의 이름, 직업, 부서번호, 부서이름을 출력
SELECT E.ENAME, E.JOB, DEPTNO, D.DNAME
FROM EMP E NATURAL JOIN DEPT D
WHERE D.LOC LIKE 'DALLAS'

5. 이름에 A가 들어가는 사원들의 이름과 부서이름을 출력
SELECT E.ENAME, D.DNAME
FROM EMP E NATURAL JOIN DEPT D
WHERE E.ENAME LIKE '%A%'

6. 사원이름과 그 사원의 관리자 이름을 출력하라  << 수정댐
(단 컬럼 HEADING을 EMPLOYEE, MANAGER로 출력)
SELECT E.ENAME AS "EMPLOYEE", C.ENAME AS "MANAGER"
FROM EMP E RIGHT OUTER JOIN EMP C
ON( E.MGR = C.EMPNO )
WHERE E.ENAME LIKE '%A%'

# 아앗 아아.. 반성, 관리자가 없을 수도 있으니 OUTER JOIN이 적합

7. 사원이름과 직업, 급여, 급여등급을 출력
SELECT E.JOB, E.SAL, GRADE
FROM EMP E JOIN SALGARDE S
ON (E.SAL BETWEEN S.LOSAL AND S.HISAL)

8. 사원이름과 부서명과 월급을 출력하는데 월급이 3000이상인 사원을 출력
SELECT E.ENAME, D.DNAME , E.SAL 
FROM EMP E, DEPT D 
WHERE E.DEPTNO = D.DEPTNO
AND E.SAL >= 3000;

SQL99 ==> SELECT E.ENAME , D.DNAME , E.SAL
	  FROM EMP E NATURAL JOIN DEPT D
	  WHERE E.SAL >= 3000; 

9. 사원이름, 부서번호와 같은 부서에 근무하는 동료사원들을 출력
SELECT C.ENAME, E.ENAME, E.DEPTNO
FROM EMP E CROSS JOIN EMP C
WHERE C.DEPTNO = E.DEPTNO
AND C.ENAME NOT LIKE E.ENAME
ORDER BY C.ENAME DESC

10. BLAKE이란 사원보다 늦게 입사한 사원의 이름과 입사일을 출력
SELECT C.ENAME, C.HIREDATE
FROM EMP E JOIN EMP C
ON (C.HIREDATE >= E.HIREDATE) #반ㄴ대아냐?
WHERE C.ENAME LIKE 'BLAKE'

# 6번은 해당 컬럼에서 NULL 값을 가지고 있을 수 있으므로 OUTER JOIN해야 된다고 생각함.
# 7, 10번은 해당 컬럼에서 NULL 값을 가지고 있을 수 없으므로 OUTER JOIN하지 않아도 된다고 생각함.
